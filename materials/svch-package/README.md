# materials/svch-package

Пакет материалов для Чемезова / Ростеха (внутри проекта — «СВЧ»). Тема: ГИС ЦТИ.

## Файлы

- **HANDOFF.md** — главный хэндофф между сессиями. Что делаем, кто Миша, что в работе, открытые развилки. Читать первым
- **canon-pptx.md** — стилистический канон боевой презентации. Палитра, шрифты, chrome, паттерны слайдов, правила добавления нового слайда. Читать перед любыми правками декка
- **BOOTSTRAP_PROMPT.md** — готовый промпт для запуска новой сессии Claude. G копипастит в чат как первое сообщение
- **binaries/** — рабочие файлы (`.pptx`, `.key`, `.xlsx`, `.pdf`). Заливаются вручную через GitHub web или через Google Drive → Claude

## Карантин имён

В любых артефактах пакета — только «Оператор», «Платформа», «Корпорация», «Заказчик», «Участник». Полный список запретов в HANDOFF.md §5.
