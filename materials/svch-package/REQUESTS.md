## Запрос 1 — ПОЛНЫЙ КАСКАД МИГРАЦИОННОГО ПОТОКА В РФ ПО ГОДАМ 2019–2026

```
Подготовь временной ряд по фактическому миграционному притоку в РФ за каждый год от 2019 до
2025 включительно (плюс прогноз/факт 2026 если есть данные). Формат — таблицы со столбцами по
годам, не агрегаты.

Период должен охватить: довоенный baseline (2019), пандемический шок (2020), восстановление
(2021), посточный отток после СВО (2022), рост патентов (2023), эффект после Крокус Сити Холл
и переориентации потока (2024), индийский разворот (2025), текущая точка (2026 1Q-2Q).

ЧАСТЬ A — ВИЗОВЫЙ КАНАЛ (квота → РнР → въезды) ПО ГОДАМ

Для каждой страны отдельная таблица 2019–2025:
— Индия, Бангладеш, Вьетнам, Шри-Ланка, Мьянма, Непал, КНДР, Куба, Иран, Турция, Сербия, Китай

Колонки таблицы:
(1) КВОТА Минтруда РФ (страновая) — ссылка на ПП о квоте
(2) РНР МВД, выданные за год (полный) — статистика МВД
(3) Фактические въезды с целью «работа» — ФСБ-Погранслужба, fedstat.ru
(4) Утилизация квоты по двум стадиям: РнР/квота%, въезды/квота%

ЧАСТЬ B — БЕЗВИЗОВЫЙ ПАТЕНТНЫЙ КАНАЛ ПО ГОДАМ

Для УЗ, ТД, Молдовы, Азербайджана, Украины — таблицы 2019–2025:
(1) Действующих патентов на конец года
(2) Выдано патентов за год
(3) Въездов с целью «работа» (ФСБ)
(4) Доля въехавших → оформивших патент (если ИЭП Гайдара / РАНХиГС / Демоскоп Weekly публикует)
(5) Поступления патентного НДФЛ в региональные бюджеты (ФНС/Казначейство, агрегатно по годам)

ЧАСТЬ C — ЕАЭС (Кыргызстан, Казахстан, Беларусь, Армения) ПО ГОДАМ 2019–2025

(1) Численность граждан, работающих в РФ (без РнР, без патента, по трудовому договору)
(2) Сколько из них оформлены как НПД-самозанятые
(3) Въездов с целью «работа» (ФСБ)
Источник: ФНС, СФР, Минтруд, ЕЭК.

ЧАСТЬ D — НАТУРАЛИЗАЦИЯ И АЛЬТЕРНАТИВНЫЕ СТАТУСЫ ПО ГОДАМ 2019–2025

(1) Выдано гражданств РФ — топ-10 стран исхода с цифрами по годам
(2) Действующих ВНЖ — топ-10 стран по годам (если МВД публикует)
(3) Действующих РВП — топ-10 стран по годам
МВД с 2023-Q3 закрыло часть этой статистики — где есть пробел, явно указать.

ЧАСТЬ E — ВКС ПО ГОДАМ 2019–2025

Численность ВКС в РФ на конец каждого года + разбивка по странам исхода (топ-10).

ТРЕБОВАНИЯ ОБЩИЕ:
— Таблицы со столбцами по годам, никаких "за период 2020-х".
— Каждая ячейка — на конкретную дату (декабрь предыдущего года или конец отчётного периода).
— Каждая цифра со ссылкой на ПЕРВИЧНЫЙ источник: МВД, ФСБ, Минтруд, ФНС, government.ru,
  pravo.gov.ru, fedstat.ru, ЕЭК.
— Если данные за конкретный год отсутствуют — явная пометка "[данные не публиковались]"
  или "[МВД закрыло публикацию с YYYY-Q]".
— Если есть только diapazon (например, "1.5–2 млн") — указывать оба значения и методологическую
  оговорку.
```

ЧАСТЬ C — ТАЙМЛАЙН РЕГИОНАЛЬНЫХ ЗАПРЕТОВ ПО ОКВЭД 2019–2026

Полный реестр по каждому субъекту РФ, который вводил/расширял запреты на патентный найм
по ст. 18.1 ФЗ-115. По каждому акту — карточка:
— Номер и дата постановления губернатора/правительства субъекта
— Список запрещённых ОКВЭД
— Дата вступления в силу
— Изменения относительно предыдущей версии (если расширяет/заменяет более ранний акт)
— Источник: pravo.gov.ru / региональный портал

Хронологическая сортировка от 2019 до сегодня. Особенно интересует динамика расширения:
сколько субъектов было в 2019, 2022, 2024, 2026.

ЧАСТЬ D — ИНОСТРАНЦЫ В ПРОМЫШЛЕННЫХ КЛАСТЕРАХ ОПК ПО ГОДАМ

ТРЕБОВАНИЯ:

## Запрос 3 — СТРУКТУРНЫЙ ПРОВАЛ ТЕКУЩЕЙ МОДЕЛИ — ДИНАМИКА ТЕНЕВОЙ ЗАНЯТОСТИ И ПАТЕНТНОГО НЕСОВПАДОСА 2019–2026

```
Тренд-сводка по тезису "текущая модель миграционного учёта (патент + свободный въезд +
ЕАЭС-канал + НПД) не обеспечивает контроль за фактической занятостью иностранцев". Нужна
динамика 2019–2025, не моментальный снэпшот.

ЧАСТЬ A — ДИНАМИКА ПОКРЫТИЯ ПАТЕНТАМИ ПО ГОДАМ

Таблица 2019–2025 для УЗ и ТД отдельно:
— Доля въехавших с целью «работа» → оформивших патент (%)
— Доля въехавших с целью «работа» → оформивших патент в первые 30 дней
— Источник: МВД ГУ по вопросам миграции, ИЭП им. Гайдара (мониторинги Флоринской), РАНХиГС,
  ВШЭ ИДЕМ, Демоскоп Weekly

ЧАСТЬ B — ПРОФЕССИОНАЛЬНЫЙ НЕСОВПАДОС ПО ГОДАМ

Свежие публикации МВД, цитаты Горового, Колокольцева, Казаковой, других замминистров с
датами выступлений. Если цифры МВД не публикует — все косвенные оценки от РАНХиГС/ВШЭ.

ЧАСТЬ C — ОЦЕНКИ ТЕНЕВОЙ ЗАНЯТОСТИ ПО ГОДАМ И ПО СТРАНАМ

Таблица 2019–2025 с верхней и нижней границами оценок:
(1) УЗ + ТД: Флоринская РАНХиГС, Чудиновских ВШЭ, Капелюшников ВШЭ, Сафонов Финуниверситет
(2) Кыргызстан + Казахстан (ЕАЭС): любые оценки занятости вне НПД и трудовых договоров
(3) Все нелегально пребывающие иностранцы: МВД, реестр контролируемых лиц с 2025

ЧАСТЬ D — РЕЕСТР КОНТРОЛИРУЕМЫХ ЛИЦ МВД (запущен 5.02.2025) ПОКВАРТАЛЬНАЯ ДИНАМИКА

Численность реестра по точкам:
— март 2025, июнь 2025, сентябрь 2025, декабрь 2025
— март 2026, июнь 2026 (если есть)

Структура реестра по причинам внесения. Цитаты Горового и других замминистров МВД с датами.

ЧАСТЬ E — РЕАКЦИЯ РЕГУЛЯТОРА ПО ГОДАМ 2019–2026

Хронология официальных заявлений и инициатив, фиксирующих провал текущей модели:
— Цитаты Котякова (Минтруд) о патентном канале — все публичные выступления 2019–2026
— Цитаты Колокольцева, Горового, Казаковой (МВД) — то же

Каждая цитата — с датой, местом выступления, точной формулировкой и ссылкой на стенограмму/

## Запрос 4 — СОГЛАШЕНИЯ ОБ ОРГНАБОРЕ И БИЛАТЕРАЛЬНЫЕ КАНАЛЫ — ВСЕ СТРАНЫ, ДИНАМИКА 2017–2026

```
Подготовь полную картину двусторонних соглашений РФ об организованном наборе иностранных
работников и фактической производительности этих каналов по годам.

ЧАСТЬ A — РЕЕСТР СОГЛАШЕНИЙ

Для каждого двустороннего соглашения РФ с третьей страной об оргнаборе/трудовой мобильности —
карточка:
— Дата подписания, дата вступления в силу
— Стороны
— Полный текст (ссылка на pravo.gov.ru / МИД / lex.uz / соответствующий первичный источник)
— Целевые сектора и квоты (если зафиксированы)
— Механизм рекрутинга, привязки к работодателю, гарантий

Минимум должны быть включены:
— РФ-Узбекистан 2017 (5 апреля 2017)
— РФ-Таджикистан (если есть, 2017?)
— РФ-Молдова (если есть)
— РФ-Индия 5.12.2025
— РФ-Бангладеш (если есть; обсуждается?)
— РФ-Шри-Ланка (если есть)
— РФ-Мьянма (если есть)
— РФ-Вьетнам (если есть)
— РФ-КНДР (исторические договорённости)
— РФ-Куба (исторические)
— РФ-Иран (если есть)

ЧАСТЬ B — ФАКТИЧЕСКИЙ ОБЪЁМ КАЖДОГО КАНАЛА ПО ГОДАМ 2017–2025

Для каждого активного соглашения — таблица:
(1) Привлечено по оргнабору за год (тыс. чел.)
(2) Общий поток граждан этой страны в РФ (для сравнения)
(3) Доля оргнабора в общем потоке (%)

Источники: Агентство внешней трудовой миграции при Минтруде Узбекистана, Минтруд Таджикистана,
Минтруд РФ, Россотрудничество, посольства соответствующих стран.

ЧАСТЬ C — СРАВНИТЕЛЬНАЯ ТАБЛИЦА УСЛОВИЙ

По всем активным соглашениям — сравнительная таблица:
— Привязка работника к работодателю (есть/нет, период)
— Привязка к региону
— Гарантии оплаты, страхования, репатриации
— Механизм проверки контрактов (одной/обеими сторонами)
— Эскроу/депозит/возвратные механизмы
— Кто оператор канала со стороны РФ (Минтруд / МВД / ФГУП / частный)

Особенно интересно: что в соглашении РФ-Индия 5.12.2025 принципиально нового по сравнению
с РФ-УЗ 2017.

ЧАСТЬ D — ЭВОЛЮЦИЯ ОРГНАБОРА В РФ КАК ИНСТИТУТ

Хронология 2017–2026:
— Когда впервые появился термин "организованный набор" в нормативных документах РФ
— Концепция Котякова (октябрь 2024) — точная дата выступления, полная цитата

— Законопроект о передаче миграционных полномочий от Минтруда к МВД (одобрен правкомиссией

— Минфин-законопроект о дифференцированном ФАП февраль 2026 — текущий статус

ТРЕБОВАНИЯ:

ОПК и платформенной экономикой РФ. Период 2019–2026 факт + прогноз 2027–2030.

ЧАСТЬ A — ДИНАМИКА ДЕФИЦИТА КАДРОВ В ОБРАБОТКЕ И ОПК ПО ГОДАМ 2019–2026

    цитаты с датами

(1) Прогноз дефицита кадров к 2030: Котяков (3.1 млн), Минтруд, HH.ru, Греф, Яков и Партнёры,
