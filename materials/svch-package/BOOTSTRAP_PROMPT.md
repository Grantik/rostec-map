## Что прикладываем drag-and-drop вместе с промптом

Файлы которые уже на маке у G — кидай прямо в чат вместе с промптом:
- `ЦТИ_ГИС.pptx` — боевая презентация (последняя версия)
- `operator-pnl-7y.xlsx` — финансовая модель Оператора
- `ЦТИ_ГИС.pdf` — PDF-экспорт боевой (опционально, если нужно чтобы Claude видел контент без открытия pptx)
- При необходимости — комментарии Миши (текст / скриншоты / отдельный документ)

Сессия с bash/filesystem (Claude Desktop, или web-чат с включёнными computer tools) сама заберёт их из uploads и зальёт в `materials/svch-package/binaries/` на гите — после этого файлы перестают зависеть от сессии.

Проект Claude уже подключён к чату: меморандум, краткая записка, юр. пакет, каскадная модель — всё видно через `<file_path>` блоки в начале сессии.
