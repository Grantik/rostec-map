## Связка моделей (без изменений после Q5/Q6)

- **Модель A (literary synthesis с цифрами .X точности)**: ВЫКИНУТА. Признаки подтверждены ещё раз на Q5/Q6:
  - Q5 V3: ошибочная привязка «Сафонова ВШЭ» (Сафонов в Финуниверситете), выдуманный «Котяков 2,4 млн обновлено», точные цифры по предприятиям без источника.
  - Q6 V2: собственные обобщения «гражданская смерть», «управляемая прозрачность», аналитические надстройки сверх запроса.
- **Модель B (numbered [N] citations / структурированные таблицы без URL)**: рабочая лошадка для контента, но **без URL непригодна по критерию 1**. На Q5 V2 даёт правильную структуру цитат Алиханова/Хоценко/Мантурова/Морозова — но требует сверки через web_fetch на kremlin.ru / council.gov.ru / duma.gov.ru.
- **Модель C (короткие отказы с прямыми URL)**: подтвердила свою роль на Q5 V1 и Q6 V3 — даёт URL-каркас, на котором держится футер презы. Содержательно скудна, но критерий 1 закрывает только она.

## Промпт для следующих запросов

Для Q4 (третий вариант для триангуляции) — жёстко требовать:
- Прямой URL первичного текста соглашения РФ-Индия 5.12.2025 ИЛИ явная фиксация «текст в открытом доступе НЕ опубликован» с URL подтверждающего источника (Joint Statement kremlin.ru / парламентский ответ MEA).
- Сверка цифры оргнабора УЗ 2024: канон 161,8 тыс. vs Коммерсантъ 123,7 тыс. — разобраться какая первичка.
- URL первичного текста РФ-Таджикистан 17.04.2019 (publication.pravo.gov.ru/Document/View/0001202002260045 — проверить).
- Подтверждение/опровержение существования соглашений с Бангладеш / Шри-Ланка / Мьянма / Куба / Иран — только подписанные акты с датами и URL, без «обсуждается».
